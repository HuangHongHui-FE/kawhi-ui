API


| 参数        | 说明                       | 类型                     | 默认值    |
| ----------- | -------------------------- | ------------------------ | --------- |
| description | 自定义描述内容             | boolean                  | false     |
| children    | 嵌套的标题                 | string                   | -         |
| className   | 分割线样式类               | default\| small \| large | default   |
| orientation | 分割线标题的位置           | start`                   | `end`     |
| size        | 间距大小，仅对水平布局有效 | small`                   | `middle`  |
| style       | 样式                       | CSSProperties            | -         |
| type        | 水平还是垂直类型           | horizontal`              | `vertical |
